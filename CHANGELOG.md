## [1.0.0] - 2025-01-12

### Added

- Initial release of voice-memos-mcp
- MCP server implementation for Voice Memos on macOS
- Support for recording new voice memos via AppleScript integration
- Support for listing existing voice memos
- Support for playing voice memos
- Model Context Protocol (MCP) SDK integration
- TypeScript support with full type definitions
- Vitest test suite
- ESLint and Prettier for code quality
- GitHub Actions CI/CD workflows
- Husky pre-commit hooks with lint-staged
